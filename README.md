## About the Application

The application is an API (Application Program Interface) for JDA Shipping Label and search for [Southstardurg.com.ph](https://southstardrug.com.ph/), that is build in [Laravel Framework](https://laravel.com/).

The application is intended for service with third party partners to use, and be integrated in thie'r own systems.
